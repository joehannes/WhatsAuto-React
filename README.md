# WhatsAuto

AI-powered WhatsApp Business desktop messaging application.

## Features

- Sidebar navigation with 10 views (Dashboard, Chats, Contacts, AI, Automation, Translation, Voice, Leads, Plugins, Settings)
- WhatsApp chat interface with contact list and message thread
- AI assistant panel with provider selection (Qwen, OpenAI, DeepSeek, Ollama)
- Contacts management with labels and search
- Automation rules and scheduled tasks
- Translation with style adaptation
- Voice (STT/TTS) configuration
- Lead discovery
- Plugin system
- Business settings and profile management

## Tech Stack

- React + TypeScript + Vite
- shadcn/ui + Tailwind CSS v4
- Supabase for data persistence
- Lucide icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
